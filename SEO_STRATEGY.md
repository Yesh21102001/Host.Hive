## 📊 SCHEMA MARKUP EXAMPLES

### Organization Schema
```json
{
  "@context": "https://schema.org",
  "@type": "LocalBusiness",
  "name": "HostHive",
  "description": "Professional web development company",
  "url": "https://hosthive.in",
  "logo": "https://hosthive.in/logo.png",
  "areaServed": {
    "@type": "City",
    "name": "Visakhapatnam"
  },
  "telephone": "+91-XXXXXXXXXX",
  "email": "hello@hosthive.in",
  "address": {
    "@type": "PostalAddress",
    "addressLocality": "Visakhapatnam",
    "addressRegion": "AP",
    "addressCountry": "IN"
  }
}
```

### Service Schema
```json
{
  "@context": "https://schema.org",
  "@type": "Service",
  "name": "Website Development",
  "description": "Professional website development services in Visakhapatnam",
  "provider": {
    "@type": "LocalBusiness",
    "name": "HostHive"
  },
  "areaServed": "Visakhapatnam"
}
```
